Linux Mint系统默认只保留当前系统版本的壁纸，我们可以使用命令安装上之前的壁纸，步骤如下：

## 1、首先更新软件列表

```shell
sudo apt-get update
```

## 2、安装壁纸

```shell
sudo apt-get install mint-backgrounds-*
```
